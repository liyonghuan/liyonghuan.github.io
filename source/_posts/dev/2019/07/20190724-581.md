```
yum install vim
```

&emsp;&emsp;博主这里选择vim而不是vi是有原因的。因为在使用过程中，vi总是有些莫名其妙的问题，体验不太好，而vim不仅符合正常的使用习惯，而且内容还会高亮显示，看代码可不要太舒服。

&emsp;&emsp;通过修改location里面的root指定nginx的根路径，在前篇[「Nginx」CentOS7.6操作系统下通过源码安装Nginx教程](https://www.klavor.com/dev/20190723-579.html)中，新增了名为www的用户并为其指定了目录/www，因此这里将root设置为/www。

### 测试效果

```
cd /www
vim index.html
```

&emsp;&emsp;进入到目录/www中并创建名为index.html的文件。

```
<html>
<head>
    <meta http-equiv="Content-Type" content="text/html;charset=utf-8">
    <title>网站正在建设中...</title>
</head>
<body>
<a href="https://www.klavor.com" target="_blank" rel="noopener noreferrer">网站正在建设中...</a>
</body>
</html>
```
