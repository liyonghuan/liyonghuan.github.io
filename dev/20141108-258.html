Linux系统之ubuntu下安装配置VPN完美教程

时间: 2014-11-08 / 分类: 开发 / 阅读时长: 1分钟 / 阅读次数: 0

之前就想要在linux系统中使用VPN了，但是不知道为什么我用的ubuntu系统就是连不上VPN。后来也就放弃了，过了Long Long Ago，我有突然心血来潮想要使用VPN下载StarDict的词库，因为好像不用VPN下不了词库。后来我在网上找到了教程，按着教程设置了以后，终于有效了。其实之前我也有百度过的，但是一直没能设置成功。下面我给出ubuntu中配置VPN的文字教程给大家，希望能够对你有帮助。

一般情况下前两步都可以省略，而且第五步应该也用不上，至少我没用上。配置好保存后，在网络连接里面选择VPN然后就会进行自动连接。

  1. 安装VPN Client

#sudo apt-get install pptp-linux

  1. 安装网络管理器对VPN的支持

#sudo apt-get install network-manager-pptp

  1. 添加VPN连接

右击面板上的网络图标->VPN连接->配置VPN

点击“添加”

选择默认的PPTP VPN连接类型，点击“建立”

连接名称随便取一个。填入网关和用户名、密码

点击“高级”，在“允许以下人证方法”中取消选中“EAP”，选上“使用点到点加密(MPPE)”

这个步骤是必须的，否则会连接不上。

确定，保存

  1. 选中建立的VPN连接就可以连上私有网络了，用ifconfig查看，会多出一个ppp0的网络接口

  2. 如果还想接着访问外网，则需要手动增加路由

route add -net 192.168.0.0 netmask 255.255.255.0 gw 192.168.0.1

这里假设192.168.0.1为网关。
