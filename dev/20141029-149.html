Linux系统下安装使用utorrent软件完整教程 BT下载工具

2014-10-29 dev BT, Liux, utorrent, 下载工具 0 评论

这些天都在找linux系统的BT下载软件，而且六维网站好像只支持utorrent这款软件。因此，我找了好久，最终下载好了，但是根据网上的教程一步一步来，最后发现为什么不行呢，直至现在我才知道原因，因为我没有把网上的教程完整的看完。

下面我给大家在linux下下载安装utorrent的完整教程。

1、下载

官方网站www.utorrent.com/downloads/linux

到官网上把这个linux的utorrent下载下来。

2、解压

tar -zxvf 【utorrent文件名】

3、进入目录

如果不知道目录的话可以按ls进行查看当前目录下的文件及文件夹

cd 【utorrent文件夹名】

4、启动utorrent

在终端输入./utserver运行utorrent。

这里可能会遇到

./utserver: error while loading shared libraries: libssl.so.0.9.8: cannot open shared object file: No such file or directory

./utserver: error while loading shared libraries: libcrypto.so.0.9.8: cannot open shared object file: No such file or directory

的问题。

可以通过解决方法为从网址www.it-adv.net/fetion/downng/library_linux.tar.gz下载包，解压后将libssl.so.0.9.8和libcrypto.so.0.9.8

复制到/usr/lib即可。

可使用代码

cp libcrypto.so.0.9.8 /usr/lib

cp libssl.so.0.9.8 /usr/lib

5、使用utorrent软件

在浏览器输入URL http://localhost:8080/gui/，输入用户名为admin，密码为空，就可以使用utorrent。下载位置为utorrent-server-VERSION文件夹。

6、关闭web UI

直接关闭网页文件会继续下载，所以直接关闭网页没用的，这是你可以使用命令killall utserver来关闭utorrent。

其实你可以直接在文件夹直接运行的，不必要输代码那么麻烦。
