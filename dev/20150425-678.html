注册StsrtSSL 免费SSL证书申请完美教程之三

时间: 2015-04-25 / 分类: 开发 / 阅读时长: 2分钟 / 阅读次数: 0

前两篇文章把申请StartSSL的SSL证书的基本步骤都准备好了，现在开始做正式的申请工作吧。其实也不是很难，主要的困难点其实就是英语啦，对于英语差差差的我来说也是极其的困难，不过我还算是比较有耐心的，通过百度翻译把这些英文全部翻译成中文后，然后按照中文上的意思然后进行了注册，最终可以说注册过程是无压力的。相信，你通过此篇文章可以连翻译都不用就可以成功申请StartSSL的免费SSL证书了。

第一步：点击第二步中出来的页面中第三个链接，Validations Wizard。这里咱们是注册网站的SSL证书，所以，这里我们选择Domain Name Validation。如图：

第二步：填写你的域名，注意选择你正确的域名后缀哦。如图：

第三步：选择你激活的邮箱，这里系统会默认给出几个一般可能会有的域名邮箱和之前注册激活StartSSL账户的时候所使用的密保邮箱，可以选择你的密保邮箱。如果你有其他的邮箱也可以选择，然后点击Continue进入下一步。如图：

第四步：打开你的邮箱，StartSSL官网的系统会发送一封带有验证码的邮件到你的邮箱，这里复制图中红框部分的代码，在下一步中会用到。

第五步：将目光再次回到StartSSL官网的申请页面中，将刚才复制的验证码粘贴到文本框内，点击下一步（Continue）。如图：

第六步：点击网页上的Finish，完成域名的设置。如图。

第七步：点击页面中中间的Certificates Wizard链接，在出来的网页中选择Web Server SSL/TLS Certigicate选择网页服务器SSL证书。如图：

第八步：输入你的密码，该密码很重要哦，之后会用于取回证书等工作。填好后点击Continue进入下一步。

第九步：页面会弹出提示框，点击确定继续。

第十步：按照页面上的要求，复制文本框内的文本，保存为ssl.key文件，文件要的编码设成ASCII格式的。然后上传到服务器，通过服务器输入命令openssl rsa -in ssl.key -out ssl.key将ssl.key转换为加密的密钥文件ssl.key。如图所示：

第十一步：选择你的域名。如下图所示：

第十二步：填写你要申请的二级域名。如下图：

第十三步：确认信息无误后，直接点击Continue进入下一步即可。

第十四步：复制文本框内的字符到新建的记事本中，保存为所有文件，使用ASCII格式保存为ssl.crt文件名。

第十五步：你已经成功完成申请啦！
