使用Eclipse便捷实现全目录文本替换

2016-05-25 dev Eclipse, 小技巧, 文本替换 0 评论

先前几天学习过程中,项目所需要的材料中有一些写死的地址难以更替,如果手动修改则将会耗费过多的时间和精力.

原本想要写段Java代码来修改的,但是由于感觉自己比较懒惰,于是也就是想想而已.后来想到了Eclipse有全文查找的功能,那么应该也会提供全文替换的功能的啊.后来找找,还真有.终于不用自己写代码来替换文本了额呵呵~~

下面是替换的步骤,其实蛮简单的,适合新手.啊哈哈~~~

第一步:随意的在Eclipse中创建一个任意类型的项目,我这里选中的是Java项目.

第二步:输入项目名称,并取消下方”Use default location”的复选框,然后自行选择要替换的文本所在的目录,并点击”Finish”确认.

第三步:选中要替换文本的项目,然后点击窗体上的菜单栏的”Search”搜索功能,在弹出的下拉列表中选中”File”选项.

第四步:在弹出的窗体中,”Containing text”表示要匹配的文本,就是你要替换前的文本.

下方的”File name patterns”是匹配文件名的,这里用*号表示匹配所有就好啦.

再往下,选中单选按钮”Enclosing projects”.

然后点击”Replace”按钮进行替换.

第五步:此时,会弹出如下图的一个确认界面,这里有两个输入栏,上方的是替换前的文本,下方是即将要替换成的文本.

在下方输入框内输入你要替换成的文本,然后点击”OK”进行替换,骚等一会便替换好啦.

看吧,是不是很简单,很方便.以后再也不用担心批量修改了.
