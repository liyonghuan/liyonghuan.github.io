世界排名网站Alexa网站信息管理教程

2015-06-03 dev Alexa, SEO, 世界排名网站, 搜索引擎, 搜索引擎优化, 教程, 网站信息管理 0 评论

世界网站排名Alexa肯定很多人都知道，但是现在这个排名也是不准确了，因为国人用得少，而且这排名还可以刷，因此，这就使得行里人都不会太过于信任这个排名了，因此也很好做Alexa的排名优化。但是对很多外行或者刚接触网站的人来说，这么排名是多么的高大上，好歹也是全球排名呀。但是这篇文章不讲如何刷Alexa全球排名，而是要讲关于Alexa中认领自己的网站并且修改该网站的基本信息的教程。

现在废话不多说，直接上教程，但是由于是国外的网站，因此访问的时候可能会出现访问不上或者是很慢的现象。如果是访问不上，那么只能用一些你懂的方法上了；如果是加载缓慢那么就忍一忍把。

第一步：登录官网www.alexa.com，如果没有帐号的可以先自行注册一个。

第二步：在首页上的“Enter a site”里输入你要认领的域名，并点击“Go”进入到网站详细页面。

第三步：拖动网页到如下图位置：

第四步：点击上图中页面上的“Edit Site Info”，然后会提示是否认领该网站，这时只要点击”Claim This Site”即可。

第五步：之后会出现如下页面，Alexa网站一共提供了三种验证方式，但是第三种方式是上锁的，貌似是需要花钱的，因此，咱们就不管了。前两种是常用的网站验证所有权的方式，第一种是上传文件到根目录，第二种是指定复制代码到网站首页。这个可以根据个人的情况选择该如何验证。

第六步：我这里选择了第一种上传文件验证所有权的方式，所以点击下图中的“Download the file”，下载文件，然后上传到网站的根目录下。

第七步：上传好之后点击页面上的“Verify the file on your site”，进行网站所有权验证。

第八步：验证好后，会得到如下图中的成功提示“Your site is successfully claimed”：

第九步：鼠标移动到你的账户名上，并点击出来的“Account Management”链接。如下图：

第十步：点击网页左侧的“Site Settings”链接,如下图：

第十一步：完成上一后，在出来的网页上会显示所有你已经认证过的网站，选择你要修改的那个网站进行信息的修改，并保存即可。

到此，所有的Alexa验证网站所有权和修改网站信息的教程就到此结束了，此时你可以通过一些站长工具查看Alexa的网站信息，这时候你可以看到你所填写的网站信息已经成功的显示了。
