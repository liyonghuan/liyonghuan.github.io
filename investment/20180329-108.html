解读网上借贷平台的利息陷阱

2018-03-29 investment 小额贷款, 消费贷, 网络借贷, 陷阱 0 评论

前言

现在市面上的网贷APP数不胜数，如网商贷、京东金条、,而各种对于这些借贷平台、借贷软件的负面新闻也随之而出。
对于我个人来说，也使用过几个比较大型的借款平台，比如支付宝里的网商贷、东方财富里的财富贷、中信信用卡里的借款以及建行的快贷。
而这些借款软件的借款利息不尽相同，有的按年来计息、有的按月来计息，有的则是按日来计息。下面我们将对这些按日计息或者按月计息以及按月等额本息还款方式中所存在的陷阱做一个比较全面的分析。

按日计息

很多人看到了借款平台上万五、万四、万三的日借款利息率，但是很多人都忽略了还款方式上的陷阱，如果你的借款的还款方式不是到期归还本息或者按月付息到期还本的方式的话，那么就要注意这里的宣称利息率了。
这里我们按照一年的借款时间进行计算，本金以12万，利息率以日息率万五来计算。

到期归还本息

  • 按年复利
    按年复利是这几种复利计算中所需归还利息最少的一种计算方式，计算方式也比较简单，只需要将日利息率乘以365天就可以得出年化利息率，然后乘以本金则可以求出一年后所需归还的利息。由图中可以看出按年复利的年利息为21900元。

  • 按月复利
    按月复利在不计算之前都可以知道是比按日复利少的，其计算方式较按日复利会简单些。需要先求出月利息率，即将日利息率乘以月份中的天数，这里为了简便以30天为一个月进行计算。则大约一个月的利息率为1.5%。加上本金的1并求得他的12次方得到的值就是一年后的本息与原始本金的比值，减去本金的1得出的是年化利息率，大约为19.56%，到期需要归还利息23474元。

  • 按日复利
    按日计息的计算方式是求得1.0005的365次方的结果即为一年后的本息与初始时本息的比值，减去1则是年利息率。由下图可以看出万五的日利息率按日复利计算，一年的利息率大约在20%，12万本金则需要归还24019元的利息。

按月计息

等额本息按月归还

这种方式就不考虑复利的计算方式了，因为计算比较麻烦，有需要可以自行计算。
下图为等额本息按月归还的计算结果：

每月的利息是1825元，一年需要支付的利息为21900元，乍一看之下好像没有什么问题，那是因为你没有仔细分析，即使我明白其中的猫腻后在给朋友计算的时候依然出现了懵逼的情况，所以就更不用说其他没有计算过的认了。
那么我们就来算算等额本息按月归还到底有多坑。
如下图所示：

由于是按月还款，所以每个月之后的可用本金是递减的，以日利息率万五计算月利息则为1.5%，那么我们可以做如下计算：

第1月 120000 * 0.015
第2月 110000 * 0.015
第3月 100000 * 0.015
第4月 90000 * 0.015
…
第11月 20000 * 0.015
第12月 10000 * 0.015

即(120000 + 110000 + 100000 + 90000 + … + 20000 + 10000) * 0.015

为了简化计算我们将上面的计算转化成：
(12 + 11 + 10 + 9 + … + 2 + 1) * 0.015 * 10000

从1一直加到12我们可以知道一共有6组13，即(12 + 1)、(11 + 2)、(10 + 3)、(9 + 4)、(8 + 5)和（7 + 6)。
也就是 13 * 6 * 0.015 * 10000
所以一年的利息应是11700元，而每个月应付的利息为975元。
但是实际上这里的等额本息按月归还每月需要支付的利息高达1825元，多了将近一倍。
也就是说实际上的利息是0.0005 * 365 * 2 = 0.365,也就是年利息率为36.5%，你说高不高？看着低那是因为每个月都还了一部分钱所以导致了到期支付的利息不高而造成了利息率不高的假象。
所以各位借款的时候还需小心其中的陷阱。

总结

其实这陷阱和日利息率没有完全的关系，和还款方式也没有完全的关系，是由于日利息率和还款方式的混合导致了陷阱的出现且不易被发现。就好比如果日利息率万五，到期归还本息，即使按日复利也才年化利息率20%，但是如果是等额本息按月还款的方式则会高达36.5%，这是很可怕的。
那么在这里建议大家在借款的时候尽量选择大厂，比如说银行，当然，银行的门槛可能会比一些五花八门的网贷平台高。否则的话，在借款的时候还需要看清条款，谨防被下套。
祝各位合理借款，生活愉快！
